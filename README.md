# React/Typescript/immer example

## Development

### Copy environment file
`cp .env-cmdrc.example.js .env-cmdrc.js`

### To find out available commands:
`yarn start`

### Start the development server

`yarn start dev`

## Test

`yarn start test`


## Test watch

`yarn start test.w`

#### Test watch

`yarn start test.w`

#### Test coverage

`yarn start test.c`

## Production

### Build

`yarn start build`

### Deploy

`yarn start build.deploy`
